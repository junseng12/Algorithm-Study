# ➕ Problem: 부분수열의 합 (BOJ 1182)

- 링크: https://www.acmicpc.net/problem/1182

## 📌 문제 요약

- 정수 N개로 구성된 수열이 주어졌을 때,
- 크기가 양수인 **부분수열** 중 원소의 합이 S가 되는 경우의 수를 구하라.

## 🔢 입력 조건

- N (1 ≤ N ≤ 20), S (-1,000,000 ≤ S ≤ 1,000,000)
- 수열의 각 원소는 절댓값 100,000 이하 정수

## 🎯 출력 조건

- 조건을 만족하는 **부분수열의 개수** 출력

## 💡 관련 개념

- 부분 집합의 모든 경우 탐색 → DFS
- 이진 분기(포함/미포함) 구조
- 예외 처리: 공집합 제외 필요 (S == 0인 경우)
